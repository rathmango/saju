# 사주 전문 분석 종합 가이드라인

## I. 기본 격국 판단

### 1. 일간 강약 정밀 판단

**1.1 계절적 강약도**
- **봄(인월~진월)**: 목 강함, 금 약함
- **여름(사월~미월)**: 화 강함, 수 약함
- **가을(신월~술월)**: 금 강함, 화 약함
- **겨울(해월~축월)**: 수 강함, 토 약함
- **교차시기(辰/戌/丑/未)**: 토 강함

**1.2 일간 강약 수치화 방법**
- 일간을 생(生)하는 오행: +2점
- 일간과 동일 오행: +1점
- 중성 오행: 0점
- 일간을 극(剋)하는 오행: -1점
- 일간이 극(剋)하는 오행: -0.5점
- 총점 +3 이상: 매우 강함
- 총점 +1~+2: 강함
- 총점 -1~0: 중간
- 총점 -3~-2: 약함
- 총점 -4 이하: 매우 약함

**1.3 육친별 강약 평가표**
```
천간 육친 강약표 (일간 기준)
---------------------------------------------
일간   | 비견/겁재 | 식신/상관 | 재성/편재 | 관성/편관 | 인성/편인
갑을(木)| 갑을(木)  | 병정(火)  | 임계(水)  | 경신(金)  | 무기(土)
병정(火)| 병정(火)  | 무기(土)  | 갑을(木)  | 임계(水)  | 경신(金)
무기(土)| 무기(土)  | 경신(金)  | 병정(火)  | 갑을(木)  | 임계(水)
경신(金)| 경신(金)  | 임계(水)  | 무기(土)  | 병정(火)  | 갑을(木)
임계(水)| 임계(水)  | 갑을(木)  | 경신(金)  | 무기(土)  | 병정(火)
```

## II. 십성(十星) 심층 분석

### 1. 비견/겁재(比劫) 심층 분석

**1.1 특성 상세**
- **비견(同性)**: 동성 형제, 경쟁자, 자신과 같은 기질의 사람
- **겁재(異性)**: 이성 형제, 협력자, 자신과 다른 성질의 조력자

**1.2 수량별 의미**
- **비견/겁재 0개**: 형제 인연 적음, 경쟁의식 약함, 독립적 성향
- **비견/겁재 1개**: 적절한 경쟁력, 균형 잡힌 협력 관계
- **비견/겁재 2개**: 강한 경쟁의식, 형제와 유대감 깊음
- **비견/겁재 3개 이상**: 과도한 경쟁, 자기주장 강함, 이기적 성향

**1.3 일간 강약에 따른 의미 변화**
- **강한 일간 + 많은 비견**: 과도한 자신감, 고집이 세다
- **약한 일간 + 많은 비견**: 지원세력 많음, 협력으로 성장
- **강한 일간 + 적은 비견**: 독자적 성취, 외로움 가능성
- **약한 일간 + 적은 비견**: 외부 지원 부족, 고립감

### 2. 식신/상관(食傷) 심층 분석

**2.1 특성 상세**
- **식신(同性)**: 자식(아들), 창의력, 표현력, 감성, 예술성
- **상관(異性)**: 자식(딸), 지적 능력, 분석력, 기술, 학습 능력

**2.2 수량별 의미**
- **식신/상관 0개**: 자녀 인연 약함, 창의력 발휘 어려움
- **식신/상관 1개**: 적절한 창의력, 자녀와 관계 원만
- **식신/상관 2개**: 강한 예술성, 자기표현 탁월
- **식신/상관 3개 이상**: 과도한 감정 표출, 자기과시 경향

**2.3 일간 강약에 따른 의미 변화**
- **강한 일간 + 많은 식신**: 창의력 뛰어남, 예술적 재능
- **약한 일간 + 많은 식신**: 에너지 소진, 감정 소모 많음
- **강한 일간 + 적은 식신**: 창의성 부족, 경직된 사고
- **약한 일간 + 적은 식신**: 자기표현 어려움, 소극적 성향

### 3. 재성/편재(財星) 심층 분석

**3.1 특성 상세**
- **정재(同性)**: 배우자(남편), 안정적 재물, 정당한 소득
- **편재(異性)**: 배우자(아내), 불안정한 재물, 부수입, 투기성 소득

**3.2 수량별 의미**
- **재성 0개**: 재물 관리 어려움, 배우자 인연 약함
- **재성 1개**: 적절한 재물 유입, 배우자 관계 안정적
- **재성 2개**: 풍부한 재물, 이성 인연 많음
- **재성 3개 이상**: 재물에 대한 집착, 사치 성향

**3.3 일간 강약에 따른 의미 변화**
- **강한 일간 + 강한 재성**: 재물 활용 능력 뛰어남
- **약한 일간 + 강한 재성**: 재물에 휘둘림, 소비 성향 강함
- **강한 일간 + 약한 재성**: 재물 부족, 근검절약 성향
- **약한 일간 + 약한 재성**: 재물운 약함, 경제적 어려움

### 4. 관성/편관(官殺) 심층 분석

**4.1 특성 상세**
- **정관(同性)**: 지위, 명예, 권위, 책임감, 조직 내 역할
- **편관/칠살(異性)**: 경쟁, 권력투쟁, 위기, 변화, 도전

**4.2 수량별 의미**
- **관성 0개**: 조직 적응 어려움, 권위 없음
- **관성 1개**: 적절한 리더십, 균형 잡힌 책임감
- **관성 2개**: 강한 권위, 사회적 성취 가능성 높음
- **관성 3개 이상**: 권위주의적 성향, 과도한 책임감

**4.3 일간 강약에 따른 의미 변화**
- **강한 일간 + 강한 관성**: 권위와 책임 균형, 리더십 발휘
- **약한 일간 + 강한 관성**: 외부 압력에 취약, 스트레스 많음
- **강한 일간 + 약한 관성**: 개인주의적 성향, 자유 추구
- **약한 일간 + 약한 관성**: 사회적 인정 부족, 목표 부재

### 5. 인성/편인(印星) 심층 분석

**5.1 특성 상세**
- **정인(同性)**: 어머니, 지식, 학문, 지혜, 보호
- **편인(異性)**: 교육자, 멘토, 조언자, 비밀, 내면의 지혜

**5.2 수량별 의미**
- **인성 0개**: 지식 습득 어려움, 모성애 부족
- **인성 1개**: 적절한 학습 능력, 모성 관계 원만
- **인성 2개**: 뛰어난 학문적 소양, 정신적 성숙
- **인성 3개 이상**: 과도한 지적 추구, 현실과 괴리

**5.3 일간 강약에 따른 의미 변화**
- **강한 일간 + 강한 인성**: 지혜로운 판단력, 학문적 성취
- **약한 일간 + 강한 인성**: 인성에 억압됨, 과도한 의존성
- **강한 일간 + 약한 인성**: 독학 성향, 자기주도적 학습
- **약한 일간 + 약한 인성**: 지식 부족, 정신적 불안정

## III. 특수 격국 분석

### 1. 일지 관계 분석

**1.1 일간-일지 합(合) 관계**
- **갑일 + 미지 = 목토합**: 안정성과 성장의 균형
- **을일 + 술지 = 목토합**: 실용적 창의성 발휘
- **병일 + 신지 = 화금합**: 예술적 표현력 뛰어남
- **정일 + 유지 = 화금합**: 섬세한 카리스마 소유
- **무일 + 자지 = 토수합**: 실용적 지혜를 갖춤
- **기일 + 해지 = 토수합**: 안정적인 직관력
- **경일 + 인지 = 금목합**: 절제된 추진력
- **신일 + 묘지 = 금목합**: 분석적 성장 능력
- **임일 + 오지 = 수화합**: 깊은 통찰력과 열정
- **계일 + 사지 = 수화합**: 유연한 적응력과 추진력

**1.2 일간-일지 충(沖) 관계**
- **갑일 + 신지 = 목금충**: 내적 갈등, 자기모순
- **을일 + 경지 = 목금충**: 이상과 현실의 괴리
- **병일 + 계지 = 화수충**: 감정과 이성의 충돌
- **정일 + 임지 = 화수충**: 열정과 냉정 사이 갈등
- **무일 + 기지 = 토토충**: 내적 불안정, 자기의심
- **경일 + 묘지 = 금목충**: 원칙과 변화 사이 고민
- **신일 + 인지 = 금목충**: 보수와 진보 사이 갈등
- **임일 + 사지 = 수화충**: 직관과 행동 불일치
- **계일 + 오지 = 수화충**: 감성과 열정 사이 혼란

**1.3 일간-일지 형(刑) 관계**
- **인일(寅日)과 해지(亥支)**: 자기비판, 완벽주의
- **진일(辰日)과 술지(戌支)**: 내적 긴장, 자기검열
- **신일(申日)과 축지(丑支)**: 철저한 자기관리, 강박적 성향

### 2. 정신살(神煞) 상세 분석

**2.1 주요 길신(吉神)**
- **천월(天月)**: 예술적 재능, 창의성 (인-묘, 오-미, 술-해)
- **천덕(天德)**: 인덕, 귀인 도움 (인-사-신, 묘-오-유, 진-술-축-미)
- **월덕(月德)**: 대인관계 원만, 복덕 (월지와의 관계)
- **복덕(福德)**: 안정, 행운 (태월 관련)
- **천의(天醫)**: 질병 극복, 건강 (자오묘유의 일간)

**2.2 주요 흉신(凶神)**
- **형살(刑殺)**: 충돌, 상처 (자형자, 축형미형술, 인형해, 사형신형진)
- **충살(沖殺)**: 급변, 단절 (자오충, 인신충, 축미충, 진술충, 묘유충, 사해충)
- **망신(亡神)**: 낭비, 손실 (월지와의 관계)
- **재살(災殺)**: 사고, 질병 (일간과 세운의 관계)
- **백호(白虎)**: 폭력성, 공격성 (일지와 유지 충돌)

### 3. 명조 격국 판단

**3.1 일간 격국**
- **건명(乾命)**: 우성 남성, 강한 아버지 기질, 지도자형
- **곤명(坤命)**: 우성 여성, 강한 어머니 기질, 포용력
- **간명(艮命)**: 침착함, 내면의 단단함, 지구력
- **태명(兌命)**: 낙천성, 사교성, 표현력 좋음
- **진명(震命)**: 결단력, 추진력, 충동성
- **손명(巽命)**: 유연성, 적응력, 창의성
- **감명(坎命)**: 지혜, 통찰력, 직관력
- **이명(離命)**: 열정, 카리스마, 표현력

**3.2 특수 격국**
- **상길격(傷吉格)**: 세 개 이상의 길신 집중
- **상흉격(傷凶格)**: 세 개 이상의 흉신 집중
- **천간일간편중격(天干一干偏重格)**: 같은 천간 4개 이상 연속
- **천간멸시격(天干滅試格)**: 일간이 모든 천간에 의해 극받음
- **한쪽오행편중격(一行偏重格)**: 한 오행이 총량의 70% 이상

## IV. 분야별 상세 운세 분석

### 1. 재물운 상세 분석

**1.1 재물 유형 구분**
- **정재형 재물**: 급여, 연봉, 월급 - 안정적 수입
- **편재형 재물**: 투자, 사업, 부업 - 변동성 있는 수입
- **식신형 재물**: 지적 재산, 예술 활동 - 창의적 수입
- **상관형 재물**: 기술, 서비스 제공 - 능력 기반 수입
- **관성형 재물**: 직위, 권한에 따른 - 지위 기반 수입

**1.2 재물 획득 방식**
- **재+인**: 지식/전문성 통한 재물 (전문직, 컨설팅)
- **재+식**: 창의력 통한 재물 (예술, 디자인)
- **재+비견**: 협력 통한 재물 (동업, 파트너십)
- **재+관**: 조직/권위 통한 재물 (공직, 기업)
- **재+살**: 위험/도전 통한 재물 (투자, 투기)

**1.3 재물운 시기별 해석**
- **월지가 재성인 시기**: 수입 증가, 재물 유입
- **대운 천간이 재성인 10년**: 장기적 재물 축적
- **연운 천간과 대운 천간이 합을 이루는 해**: 재물 기회 극대화
- **월주에 파군(破軍)이 있는 달**: 지출 증가, 재물 유출
- **세운에 탐랑(貪狼)이 있는 해**: 재물 욕심 강해짐

### 2. 건강운 상세 분석

**2.1 체질별 건강 특성**
- **목체질**: 간, 담낭, 신경계 약함 / 스트레스에 취약
- **화체질**: 심장, 혈관계 약함 / 혈압 문제 주의
- **토체질**: 소화기관, 췌장 약함 / 대사성 질환 주의
- **금체질**: 폐, 호흡기 약함 / 호흡기 질환 주의
- **수체질**: 신장, 방광 약함 / 체액 불균형 주의

**2.2 오행 과부족과 건강**
- **목 과다**: 간 기능 항진, 고혈압, 두통
- **목 부족**: 근육 약화, 신경계 불안정
- **화 과다**: 염증성 질환, 발열, 심장 부담
- **화 부족**: 혈액순환 저하, 저체온, 무기력
- **토 과다**: 비만, 부종, 소화불량
- **토 부족**: 영양흡수 장애, 위장 약화
- **금 과다**: 경직, 건조함, 피부 문제
- **금 부족**: 면역력 저하, 호흡기 약화
- **수 과다**: 신장 부담, 부종, 체액 과다
- **수 부족**: 탈수, 노화 가속, 관절 문제

**2.3 위험 시기 판단**
- **일간을 극하는 대운**: 만성질환 주의 시기
- **대운/세운 모두 일간 극할 때**: 급성질환 주의
- **세운에 병부(病符)가 있는 해**: 질병 발병 가능성
- **충살(沖殺)이 집중되는 시기**: 사고 주의
- **공망일(空亡日)의 활동**: 집중력 저하, 사고 위험

### 3. 연애/결혼운 상세 분석

**3.1 배우자 유형 예측**
- **정재(正財) 강함**: 안정적, 책임감 있는 배우자
- **편재(偏財) 강함**: 자유분방하고 매력적인 배우자
- **식신(食神) 강함**: 창의적이고 감성적인 배우자
- **상관(傷官) 강함**: 지적이고 분석적인 배우자
- **정관(正官) 강함**: 권위 있고 책임감 강한 배우자
- **편관(偏官) 강함**: 도전적이고 카리스마 있는 배우자
- **정인(正印) 강함**: 지혜롭고 배려심 깊은 배우자
- **편인(偏印) 강함**: 신비롭고 독특한 매력의 배우자

**3.2 결혼 시기 예측**
- **재성(財星)이 대운에 들어오는 시기**: 인연 만날 확률 높음
- **합(合)이 이루어지는 연운**: 결혼 시기 암시
- **월주에 합(合)이 있고 연운에 합(合)이 맞물릴 때**: 결정적 만남
- **일간과 시지(時支)가 합(合)인 경우**: 운명적 만남 가능성
- **연주와 월주가 합(合)인 경우**: 이성 복이 많은 사람

**3.3 관계 질적 분석**
- **일간-재성 이상적 관계**: 상생(相生) 관계일 때 조화로움
- **일간-재성 상극(相剋) 관계**: 갈등 가능성 높음
- **재성 주변 십성 환경**: 제3자 개입 가능성 암시
- **일지-시지 관계**: 부부의 궁합 암시
- **칠살(七殺) 위치와 강도**: 이별/이혼 가능성 암시

### 4. 직업운/사업운 상세 분석

**4.1 적합 직업군**
- **목일간**: 교육, 출판, 농업, 환경, 텍스타일
- **화일간**: 요식업, 엔터테인먼트, 마케팅, 심리상담
- **토일간**: 부동산, 건설, 행정, 서비스업, 유통
- **금일간**: 금융, 법률, IT, 공학, 보안, 품질관리
- **수일간**: 연구, 무역, 여행, 해운, 물류, 어업

**4.2 직업 성취 패턴**
- **관성 강한 사주**: 조직 내 성공, 지위 상승
- **식신 강한 사주**: 창의적 분야 두각, 자유직
- **인성 강한 사주**: 학문, 연구 분야 성취
- **비견 강한 사주**: 독립적 사업가, 경쟁 분야
- **재성 강한 사주**: 재물 관리, 영업 성공

**4.3 경력 변화 시기**
- **대운 변화 시점**: 큰 직업 전환점
- **관성/인성이 들어오는 대운**: 승진, 인정받는 시기
- **칠살이 강해지는 시기**: 직업적 도전, 갈등
- **공망(空亡)이 겹치는 해**: 진로 불안정, 방향 상실
- **합(合)이 이뤄지는 해**: 새로운 기회, 협력 강화

## V. 운세의 시기별 분석

### 1. 대운(大運) 심층 분석

**1.1 대운 진행 방향**
- **양간 남명/음간 여명**: 순행 대운
- **음간 남명/양간 여명**: 역행 대운

**1.2 대운 영향력 평가**
- **원국과 대운의 상호작용 수치화**:
  - 용신의 대운: +2점
  - 희신의 대운: +1점
  - 중성신의 대운: 0점
  - 흉신의 대운: -1점
  - 원국에 없던 오행 보충: +0.5점
  - 원국 과다 오행 추가: -0.5점

**1.3 대운 전환기 특성**
- **대운 시작 초반 3년**: 전환기, 적응 시기
- **대운 중반 4년**: 안정기, 성취 시기
- **대운 마지막 3년**: 마무리, 다음 대운 준비

### 2. 연운(流年) 심층 분석

**2.1 세운(歲運) 상세 판단**
- **대운과 연운 천간 조합 분석**:
  - 천간 합(合): 기회, 인연, 변화
  - 천간 충(沖): 급변, 도전, 위기
  - 천간 형(刑): 스트레스, 갈등, 성장
- **연간 특이사항**: 태세(胎歲), 양저(養雌), 관대(冠帶), 천이(遷移), 허울(虛耗), 대사(大祀)

**2.2 삼합/육합 운기**
- **삼합(三合) 형성**: 인-오-술 / 해-묘-미 / 자-신-진 / 축-사-유
  - 대운-세운-원국 삼합 시: 최상의 운기
  - 대운-세운 부분 삼합: 좋은 기회
- **육합(六合) 형성**: 자-축, 인-해, 묘-술, 진-유, 사-신, 오-미
  - 대운-세운 육합: 협력 강화, 조화

**2.3 귀문관지(鬼門關至)**
- **운의 최고점**: 천간 합(合) + 지지 합(合) 동시 발생
- **운의 최저점**: 천간 충(沖) + 지지 충(沖) 동시 발생

### 3. 월운/일운 심층 분석

**3.1 월운 특성**
- **생월(生月)**: 매년 자신의 생월은 에너지 재충전 시기
- **절기(節氣)에 따른 월운 변화**: 24절기 흐름에 따른 기운
- **월건(月建)과 원국 관계**: 월 특성과 사주 상호작용

**3.2 일운 중요 포인트**
- **삼합일(三合日)**: 자신의 사주와 삼합 관계 형성 시 행운의 날
- **육합일(六合日)**: 사주와 육합 관계 형성 시 협력 좋은 날
- **충살일(沖煞日)**: 사주와 충돌 관계 형성 시 주의해야 할 날
- **천덕일(天德日), 월덕일(月德日)**: 좋은 일 시작하기 좋은 날

## VI. 전문가급 종합 분석 방법

### 1. 중화(中和) 법칙 적용

**1.1 상극 관계의 중화**
- **목극토, 토생금**: 목이 토를 극하나 토가 금을 생하면 중화
- **화극금, 금생수**: 화가 금을 극하나 금이 수를 생하면 중화
- **수극화, 화생토**: 수가 화를 극하나 화가 토를 생하면 중화
- **금극목, 목생화**: 금이 목을 극하나 목이 화를 생하면 중화
- **토극수, 수생목**: 토가 수를 극하나 수가 목을 생하면 중화

**1.2 삼각 관계 중화**
- **A극B, B극C, C극A**: 세 오행이 서로 균형 이루어 중화
- **예: 목극토, 토극수, 수극목** - 삼각 균형 관계

**1.3 용신(用神) 조합 중화**
- **일간 생 용신, 용신 생 X**: 연결된 오행 사이 중화
- **흉신과 길신의 병립**: 서로 상쇄 효과 발생

### 2. 특수 격국 해석 방법

**2.1 겁재격(劫財格)**
- **특징**: 비견, 겁재 과다한 사주
- **장점**: 강한 독립성, 경쟁력, 자존감
- **단점**: 협력 어려움, 고집, 대인관계 갈등
- **흐름 관찰**: 재성 또는 관성이 강한 대운에서 완화

**2.2 재고격(財庫格)**
- **특징**: 재성 풍부, 일간 재성 관계 조화로운 사주
- **장점**: 재물복 강함, 경제적 안정
- **단점**: 재물 집착, 물질주의적 성향
- **흐름 관찰**: 인성 또는 식신이 강한 대운에서 균형

**2.3 관성격(官星格)**
- **특징**: 관성 풍부, 일간-관성 관계 조화로운 사주
- **장점**: 리더십, 조직 적응력, 사회적 인정
- **단점**: 권위주의, 과도한 책임감
- **흐름 관찰**: 식신 또는 비견이 강한 대운에서 균형

**2.4 인성격(印星格)**
- **특징**: 인성 풍부, 일간-인성 관계 조화로운 사주
- **장점**: 학문적 재능, 지혜, 교육 능력
- **단점**: 현실과 괴리, 과도한 이상주의
- **흐름 관찰**: 재성 또는 관성이 강한 대운에서 균형

### 3. 신살(神煞) 중첩 분석법

**3.1 길신(吉神) 중첩**
- **명월주천(明月珠天)**: 2개 이상 길신 중첩 시 특별한 행운
- **복덕+천덕 중첩**: 인덕과 복덕 겸비, 사회적 신망
- **문창+문곡 중첩**: 학문적 성취, 시험 합격

**3.2 흉신(凶神) 중첩**
- **상문(喪門)+조객(弔客)**: 이별, 단절, 상실
- **병부(病符)+사부(死符)**: 건강 악화, 중병
- **탐랑(貪狼)+함지(咸池)**: 감정 문제, 삼각관계

**3.3 길흉 상쇄 법칙**
- **삼합(三合)과 충살(沖殺)**: 삼합이 충살의 부정적 영향 감소
- **천덕(天德)과 재살(災殺)**: 천덕이 재살의 위험 감소
- **월덕(月德)과 망신(亡神)**: 월덕이 망신의 손실 감소

## VII. 사주 분석 종합 실전 가이드

### 1. 분석 순서 가이드

**1단계: 일간 파악과 강약 판단**
- 일간 확인 및 계절적 강약 평가
- 일간 주변 오행 관계 분석
- 일간 강약 수치화

**2단계: 용신과 격국 파악**
- 일간 기준 오행 균형 확인
- 부족/과다 오행 파악
- 용신 및 희신 설정
- 사주 격국 판단

**3단계: 천간-지지 관계 분석**
- 일주-다른 주 관계 확인
- 합(合), 충(沖), 형(刑), 해(害) 관계 파악
- 특별한 조합 패턴 발견
- 핵심 주(柱) 관계 파악

**4단계: 십성 배치 분석**
- 각 십성의 수량과 강약 평가
- 육친 관계 설정
- 핵심 육친 영향 파악

**5단계: 대운 분석**
- 대운 방향 및 특성 파악
- 대운-원국 상호작용 분석
- 기회와 위기의 대운 구분

**6단계: 세운 분석**
- 현재/미래 세운과 대운 조합
- 특별한 운기 패턴 발견
- 중요 시기 표시

**7단계: 종합 해석**
- 핵심 강점과 약점 정리
- 인생 주요 시기 예측
- 개인 맞춤형 조언 도출

### 2. 실전 팁

**2.1 경험적 해석 가이드**
- 사주의 90%는 대운으로 결정된다고 가정
- 현재 상황은 대운 70% + 세운 30%로 판단
- 날짜 선택은 월운 50% + 일운 50%로 판단
- 모든 오행은 살아있는 에너지임을 명심
- 인생의 흐름은 항상 변화하고 있음을 이해

**2.2 상담 테크닉**
- 핵심 내용을 쉬운 말로 설명
- 너무 부정적인 내용은 완화하여 전달
- 긍정적 측면과 부정적 측면 균형있게 전달
- 운명은 변화 가능함을 강조
- 미래 행동에 대한 실용적 조언 제공

**2.3 오류 방지 체크리스트**
- 음양력 변환 정확히 확인
- 출생 시간 정확성 검증
- 대운 시작점 계산 재확인
- 오행 개수 합계 재확인 (천간 4개, 지지 4개)
- 십성 관계 설정 정확히 확인

이 상세 가이드라인은 사주 분석의 깊이와 정확성을 높이는 데 도움이 될 것입니다. 사주는 단순한 점술이 아닌 동양 철학과 우주 법칙의 응용임을 이해하고, 개인의 성장과 발전을 위한 지혜의 도구로 활용하시기 바랍니다.
